Avafli is a product and consulting agency specializing in sweepstakes, gamification, marketing technology, and ad mediation. With 75+ years of combined expertise across marketing, gaming, creative design, and data management, we build the tools that drive user engagement at scale.

## WINR SDK

**Sweepstakes & engagement infrastructure for mobile app publishers.** Integrate in two lines of code — unlock daily streaks, prizing, and gamification that keep users coming back.

| Platform | Package |
|----------|---------|
| iOS | [winr_ios_sdk](https://github.com/AVAFLI/winr_ios_sdk) — Swift · SPM |
| Android | [winr_android_sdk](https://github.com/AVAFLI/winr_android_sdk) — Kotlin · JitPack |
| Flutter | [winr_flutter_sdk](https://github.com/AVAFLI/winr_flutter_sdk) — Dart · pub.dev |
| Web | [winr_web_sdk](https://github.com/AVAFLI/winr_web_sdk) — TypeScript · npm |

### Key Features

- **Sweepstakes Engine** — Compliant prizing and reward mechanics out of the box
- **Daily Streaks** — Configurable streak-based engagement loops
- **Gamification** — Points, levels, and achievements to drive retention
- **Ad Mediation** — Monetization tools built into the engagement layer
- **Two-Line Integration** — Drop-in SDK across all major platforms
